# Santa's Workshop Management System

Lightweight FastAPI project to manage toys, elves, and orders.

See `PROJECT_DOCUMENTATION.txt` for full build instructions and technical details.

Quick start:

```powershell
pip install -r requirements.txt
uvicorn main:app --reload
# open http://127.0.0.1:8000
```

To push to GitHub:

1. Create a new empty repository on GitHub (do not initialize with README).
2. Run the following from this project root:

```bash
git remote add origin <your-remote-url>
git branch -M main
git push -u origin main
```
